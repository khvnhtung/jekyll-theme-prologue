Werewolf village generator

This is a village generator for Werewolf, the deception-type role-playing game. For more background info, I thoroughly recommend Max Temkin's writeup. Hover over the colored circles with the role values to get a little information about each role.

Based on the UWUE list and this spreadsheet. The Wikia page is also a great resource.

Instructions

  1. Fill out the desired Number of players (Moderator not included) and Minimum percent plain villagers fields.
  2. For every role, choose if you would like it to be allowed in the game or not. All selected evil roles will always be included. Hover over a role's badge to read it's description.
  3. Click generate to generate a new village.
